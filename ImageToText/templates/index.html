Image-to-Text: Upload & Describe







The Flask Image Upload and Description project is designed to provide a seamless solution for handling image uploads, generating textual descriptions for images, and displaying uploaded images along with their descriptions. This project utilizes Python and Flask for server-side development, along with the Hugging Face transformers library for implementing the image-to-text pipeline.



Upload an Image

Features

  • Image Upload and Text Generation: Our website allows users to easily upload images, which are then processed using the Hugging Face image-to-text pipeline to generate descriptive text for each image. The generated text is stored in a SQLite database, enabling efficient retrieval and management of image data.
  • Display Uploaded Images and Texts: Using our website, users can retrieve and view all uploaded images along with their associated text descriptions. The API provides a clear and structured response, making it easy to access and display image data in applications or websites.
  • Efficient Text Retrieval for Repeated Images: As a bonus feature, our API includes functionality to efficiently handle repeated image uploads. By checking for duplicate images based on file paths or content hashes, the API can retrieve and return stored text descriptions from the database for duplicate images, eliminating the need for redundant processing.
